# ALONE CODE STUDIO v7 🚀

**Fully Offline Professional Web IDE**

## Run Locally
```bash
# Python
python3 -m http.server 8080

# Node.js  
npx serve .

# Then open: http://localhost:8080
```

## Deploy to GitHub Pages
1. Push this folder to a GitHub repo
2. Go to Settings → Pages → Deploy from branch `main`
3. Visit `https://github.com/Akarshana0/aload-code-studio-v7.git`

## Features
- ⚡ All 20 languages run **offline** (simulators)
- 🌐 Online → full cloud execution via Piston API
- 📁 **Import files** from device storage, paste, or URL
- 📱 PWA — installable on iOS, Android, Windows, Linux
- 🔀 Split view editor
- 🤖 AI Assistant (Anthropic Claude, online)
- 8 color themes, 4 fonts

## Keyboard Shortcuts
| Key | Action |
|-----|--------|
| Ctrl+Enter | Run code |
| Ctrl+S | Save |
| Ctrl+I | Import files |
| Ctrl+F | Find/Replace |
| Ctrl+B | Toggle sidebar |
| Ctrl+\\ | Split view |
